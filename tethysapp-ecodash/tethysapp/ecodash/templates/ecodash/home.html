Help

This applications provides a near real-time method for quantifying vegetation using cloud computing technology and remote-sensing data. This application uses a novel custom indicator to monitor vegetation on a planetary scale, and then combined this with remote-sensing data for near real-time customized quantification of vegetation change. Google Earth Engine, a cloud-based geo-computational technology, is used to temporally, geographically and adaptively filter data collections, all while performing calculations on a global scale. These technologies can help policy makers, project managers and other non-experts to quantify vegetation for various purposes including monitoring and ecosystem valuation, and allow them to use the results for local economic and social progress in a developing nation. To read more about the application pleas read the following paper: An Operational Before-After-Control-Impact (BACI) Designed Platform for Vegetation Monitoring at Planetary Scale.

To get started:

  1. Select your before intervention period range
  2. Select your after intervention period range
  3. Select the geographic region of the control plot
  4. Select the geographic region of the intervention plot
  5. Retrieve the data from analysis by:
    • Display the analysis data using the "Show time series" button
    • Or, download the geographic data for individual plots using the "Download" buttons
